# multiple_google_forms
webpage for submitting multiple google forms

# how to use
1. edit FORMID to your google form ID
```html
        const FORMID = "";
```
2. edit entry # to your google form entry #
```html
        const ENTRY = {
            name: "entry.",
            bill: "entry.",
            card: "entry.",
            cache: "entry."
        };
```
